# PRD: ルーマンのオートポイエーシスシミュレーションの初心者向け解説化 & リアル人間アバター会話UI/UX強化

## 概要と目標
難解なルーマンの社会システム理論（オートポイエーシス、二元コード、作動的閉鎖、構造的結合など）を、事前知識がない普通の人・初心者でも一目で直感的に理解できるようにUI/UXを抜本的に刷新します。
また、エージェントを文字や無機質なアイコンではなく、「実際の人間のようなリアリスティックなアバター」として視覚化し、人間同士が対話・討論しているかのような没入感のある対話ステージUI/UXを提供します。

## ターゲットユーザー / ユースケース
- 社会学やシステム理論の予備知識を持たない一般ユーザー・学生
- オートポイエーシス論やマルチエージェント社会シミュレーションのデモ・プレゼンを行う研究者・教育者
- 人工知能エージェント間の社会形成プロセスを視覚的に楽しみたいユーザー

## 機能要件 (必須)

### 1. 初心者向け解説・かみくだくモード (ELI5 / Layperson Accessibility)
- **「初心者モード（かみくだく解説）」切り替えトグル**:
  ヘッダーにトグルスイッチを配置し、ON時に専門用語を日常語に置き換え＆補足説明を表示。
- **エージェント発言の自動平易化・1行要約 (ELI5 Summary)**:
  難解なシステム論的発言に対し、初心者向けの「かみくだく要約」バッジ・吹き出しを自動併記。
- **ルーマン理論 用語翻訳＆ライブガイド（「ルマン先生のやさしい解説」）**:
  - `二元コード (Binary Code)` ➔ `【判断のものさし】(例: 法/違法, 得/損, 真/偽)`
  - `作動的閉鎖 (Operational Closure)` ➔ `【ルール内での自己増殖】`
  - `構造的結合 (Structural Coupling)` ➔ `【他者からの刺激への独自反応】`
  - `環境の摂動 (Irritation)` ➔ `【外部からのニュース・刺激】`
  シミュレーションの進行に合わせて、今何が起きているかをリアルタイムで解説する対話ガイドパネルを提供。

### 2. リアル人間風アバター & 対話ステージ UI/UX (Human Avatar & Conversation Stage)
- **リアリスティックな人間アバター定義 & ポートレート**:
  各社会システム（法、経済、科学、政治、メディアなど）に実在感のある人間ペルソナ（弁護士、経営者、科学者、政治家、ジャーナリスト）を割り当て、高品質なポートレートアバター・役職・背景色を設定。
- **円卓・討論ステージ表示モード (Avatar Roundtable Stage)**:
  従来のネットワークグラフに加え、人間アバターたちがテーブルを囲んで対話している「円卓討論ステージ (Roundtable View)」を新規追加。
- **リアルな会話演出**:
  - 発言中アバターのスポットライト強調＆リップシンク/音声波形アニメーション
  - 次の発言者の思考インジケーター（「思考中…」吹き出し）
  - アバター発言時の吹き出し（メッセージバブル）のアニメーションと表情フィードバック
- **詳細アバターカード モーダル**:
  アバターをクリックすると、その人物の専門・考え方・過去の発言・「独自の判断基準」を平易な言葉で説明する詳細カードを表示。

## 非機能要件 (パフォーマンス、UX等)
- バックエンドの API 互換性を保ち、既存の WebSocket/SSE 通信およびデータモデルと完全連動。
- レスポンシブで滑らかなアニメーション（60fps SVG/CSS transition）。
- `mypy --strict`, `npx tsc -b`, `npm run lint`, `pytest`, `vitest` 全てがエラーゼロでパスすること。
- `./.shared-agents/harness/verify.sh` が 100% 成功すること。

## 受け入れ基準 (Acceptance Criteria)
- [x] 初心者モード（ELI5 Mode）のトグル切り替えができ、オン時に専門用語が日常語解説付きで表示されること。
- [x] 発言メッセージに初心者向けのかみくだく要約が表示されること。
- [x] エージェントがリアルな人間風アバター（ポートレート、役職、カラーグラデーション、ステータス）として描画されること。
- [x] 円卓討論ステージ（Roundtable Stage）でアバター間のライブ会話演出（スポットライト、波形風アニメーション、吹き出し）が動作すること。
- [x] アバターをクリックして詳細な人間ペルソナ情報・判断基準の解説カードが閲覧できること。
- [x] Docker 環境 (`docker compose up`) およびローカル環境で `./.shared-agents/harness/verify.sh` がパスすること。
