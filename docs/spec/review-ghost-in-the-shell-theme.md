## ペルソナ別ハイライト
- **Architect**: 既存のコンポーネントツリーやデータ構造を変更せず、Tailwind カラーの拡張と SVG・CSS のみでテーマを完全に差し替えているため、極めて安全で影響範囲が局所化された設計。
- **Security**: 静的アセット、CSS、SVG のみの変更であり、セキュリティへの影響や攻撃サーフェスの拡大は一切ない。
- **Performance**: アニメーションには軽量な CSS および Framer Motion の SVG アニメーション（pathLength / scale / rotate）を用いており、GPU アクセラレーションが効くため、実行時パフォーマンスへの影響は皆無。
- **QA**: コンポーネントのテストで使用されているセレクタ（`aria-label` や `role` 等）には変更がなく、既存の機能テストやスナップショットが壊れる懸念はない。
