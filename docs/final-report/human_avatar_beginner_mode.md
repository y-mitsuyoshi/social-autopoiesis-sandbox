# 最終レポート: ルーマン・オートポイエーシス一般向け解説 & リアル人間アバター対話UI/UX

## 概要
ユーザーからの要請：「ルーマンのオートポイエーシスの実験なのですが普通の人や事前知識がない人が見てもわからないのでそういった人にもわかるようにしましょう。またUI/UXで実際の人間のようなアバターが会話しているような感じにしましょう。」に応え、以下の新機能およびUI/UXの抜本的拡張を完了しました。

## 開発成果物一覧

### 1. 初心者向け「かみくだく解説」モード (ELI5 Mode)
- **用語翻訳エンジン (`frontend/src/lib/eli5Translator.ts`)**:
  - `二元コード` ➔ `【判断のものさし】(例: 法/違法, 得/損, 真/偽)`
  - `作動的閉鎖` ➔ `【ルール内での自立思考】`
  - `構造的結合` ➔ `【外部刺激への独自反応】`
  - `オートポイエーシス` ➔ `【会話が会話を生む自己増殖】`
- **1行対話要約 (`getEli5MessageSummary`)**:
  専門的で難解なLLMエージェントの発言に対し、「〇〇さんが自身の専門の物差しで何を判断しているか」を1文で日常語要約した吹き出しバッジを表示。
- **ヘッダートグルスイッチ (`💡 初心者かみくだく解説: ON/OFF`)**:
  ユーザーがいつでも専門モードと初心者モードをワンクリックで切り替え可能。

### 2. 「ルマン先生のやさしい解説ガイド」パネル (`LuhmannTeacherPanel.tsx`)
- ルーマン本人のアバターが会話をリアルタイム実況する「実況ライブ」タブ。
- 専門用語を日常の例えで説明する「用語かみくだく辞典」タブ。
- 「なぜ人間ではなく会話が主役なのか？」などの「よくある疑問 (Q&A)」タブ。

### 3. リアリスティック人間アバターシステム (`HumanAvatar.tsx` & `humanPersonas.ts`)
- **専門分野ペルソナ定義**:
  - 法システム: 「神楽坂 律」(弁護士・法理論研究者)
  - 経済システム: 「財前 晃」(CFO・最高財務責任者)
  - 科学システム: 「Dr. 芹沢 玲」(首席科学研究員)
  - 政治システム: 「政岡 巧」(政策アナリスト)
  - メディアシステム: 「新島 杏」(ジャーナリスト・編集長)
  - ガイド: 「ルマン先生」(社会学者)
- ベクター/SVGポートレートイラスト、役職バッジ、発言中のリアルタイム音声波形・光のオーラ、思考中アニメーションを搭載。

### 4. 円卓対話ステージ UI (`RoundtableStage.tsx`)
- エージェントが会議テーブルを囲んで対話しているインタラクティブな2Dステージ（`🗣️ 人間アバター対話` モード）。
- 発言中のアバターからダイナミックに伸びる吹き出し、中心テーブルでのリアルタイム要約表示。

### 5. 詳細アバターカード モーダル (`AvatarDetailModal.tsx`)
- アバターをクリックすると、その人物の専門・考え方・過去の発言・初心者向け「判断のものさし」解説が表示されるモーダルカード。
