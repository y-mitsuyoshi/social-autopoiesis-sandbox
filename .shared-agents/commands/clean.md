---
description: 未使用コード・型エラー・リント違反を一括で洗い出す
agent: build
---

対象: $ARGUMENTS （未指定の場合はリポジトリ全体）

implementer-reviewer ペルソナとして、コードベースのクリーンアップ対象を洗い出してください：
1. **Python**:
   - `ruff check . --select F401,F811,F841,ARG` で未使用 import / 未使用変数 / 未使用引数を検出
   - `mypy .` で型エラーを列挙
2. **TypeScript/React**（存在する場合）:
   - `npm run lint` の警告を列挙
   - `npx tsc -b --noEmit` で型エラーを列挙
3. **未使用ファイル**: 参照されていないモジュール・コンポーネントを推定
4. 発見項目をファイル:行番号付きでリスト化し、安全に削除/修正できるものと要注意なものを分けて表示
5. 自動修正は行わず、リストのみ出力すること
