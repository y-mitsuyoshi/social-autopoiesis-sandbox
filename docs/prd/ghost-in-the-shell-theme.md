# PRD: Ghost in the Shell HUD UI Theme Redesign

## 概要と目標
フロントエンドのビジュアルおよびUI/UXテーマを「攻殻機動隊（Stand Alone Complex / 公安9課）」のタクティカルHUDや電脳通信をモチーフとしたデザインに刷新する。
他の機能追加や変更を行わずに、純粋なビジュアル面の強化のみを実施し、現在開発中の別エージェント等と整合性を保ちながら既存機能やテストコードとの100%の互換性を保証する。

## ターゲットユーザー / ユースケース
- 没入感のある近未来的で洗練されたビジュアルでシミュレーションを行いたいユーザー

## 機能要件 (必須)
- [ ] カラーテーマを「サイバーグリーン」から、攻殻機動隊のイメージに合う「タクティカルアンバー/オレンジ（#ff9d00）」、「サイバーシアン/ブルー（#00f0ff）」、「スチールダークブルー（#060913）」の組み合わせに変更する。
- [ ] タイトルヘッダー部を「SECTION 9 SECURITY PROTOCOL // DIRECT NEURAL CONNECTION」などのデカール風文言を含んだレイアウトに刷新し、アニメーション（pulse、ping）を付与する。
- [ ] HUDパネル（`.hud-panel`）の角括弧（コーナーブラケット）をアンバーオレンジに変更し、より頑健な見た目にする。
- [ ] `NetworkGraph` の背景に、回転する円形レーダースイープアニメーション、同心円グリッド、および角度目盛り（ターゲットレティクル）を追加する。
- [ ] 発言中のエージェント（`currentSpeaker`）の周囲に、追尾型ターゲットブラケット（4つ角のオレンジ色のフレーム）および回転する破線リングを描画し、電脳によるハッキング/通信ターゲティング風の演出を追加する。

## 非機能要件 (パフォーマンス、UX等)
- UI/UX は攻殻機動隊の独特な電脳・公安9課の雰囲気を再現したプレミアムで美しいものにする。
- 新規追加した SVG アニメーションや CSS グラデーションが軽量で、描画遅延やCPU負荷を発生させないこと。

## 受け入れ基準 (Acceptance Criteria)
- [ ] ヘッダーやパネル、グラフ内の配色が、サイバーシアンおよびタクティカルオレンジを基調としたカラーになっていること。
- [ ] `NetworkGraph` の中心を基準としたレーダースイープが円形に回転し続けること。
- [ ] アクティブエージェントの周りに専用のターゲットフレームが描画されること。
- [ ] 全自動テスト（ruff、mypy、pytest、ESLint、tsc、Vitest）がすべて正常にパスすること。
- [ ] Docker 環境 (`docker compose up`) で正常に起動すること。
