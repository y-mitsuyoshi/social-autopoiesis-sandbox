project-manager ペルソナとして、現在の開発状況をレポートしてください：
1. `docs/prd/`, `docs/spec/`, 直近の git ログ、未完了の todo を確認
2. 進行中のフェーズ（PRD/設計/実装/レビュー/QA/リリース）を特定
3. ブロッカー・遅延・リスクを明示
4. 次に取るべきアクションを3つまで優先度順に提示

`.shared-agents/templates/status.md` があればその形式に従うこと。
