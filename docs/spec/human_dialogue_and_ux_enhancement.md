# Tech Spec: 人間らしい対話化・完了演出・ルーマン考察＆ガイド強化・UIチカチカ改善

## 設計方針

### 1. AI対話プロンプトの改修 (`presets.ts`, `simulation.py`, `agents-*.yaml`)
- `simulation.py` の `guideline` 補足プロンプトを改修：
  - 論文形式（`### 1. 前提コード` や過度なマークダウン表・箇条書き）の出力を明示的に禁止。
  - 「相手の発言への反応」「問いかけ」「人間らしい対話的語り口」を要求。
- 各システム（経済・科学・法・芸術・メディア・政治・教育）のプロンプトを「人間らしさ」を保ちつつ二項コードを反映した発話スタイルに調整。

### 2. シミュレーション完了状態の修正と完了演出 (`App.tsx`)
- `agentsWithState` の計算ロジック修正：
  - `status === "completed" || status === "failed"` の場合、すべてのエージェントの `state` を確実に `"idle"` に設定。
- シミュレーション完了時に表示する演出メッセージコンポーネントを追加。

### 3. ルーマン社会システム論 総合考察セクションの拡張 (`AnalysisPanel.tsx`)
- シミュレーション完了時に対話ログ全体の構造的結合とオートポイエーシス作動について、定量データ＋学術的考察文章を自動生成・表示するロジックを拡張。

### 4. ルマン先生ガイドの大幅コンテンツ拡張 (`LuhmannTeacherPanel.tsx`)
- ルーマン理論解説（オートポイエーシス、二元コード、作動的閉鎖、構造的結合、心理システムと社会システム）の文章量を2倍以上に拡充。
- Q&A項目を追加（「なぜ人間ではなく『コミュニケーション』が主役なのか？」「どうして社会システムは他者の言葉をそのまま受け取らないのか？」など）。

### 5. UIの点滅（チカチカ）修正
- `GlitchText.tsx`: 高速な文字ブレ/色ブリンクアニメーションを静的なサイバーフロンティア風タイポグラフィに変更。
- 各種コンポーネント（`App.tsx`, `RealtimeCyberMetrics.tsx`, `RoundtableStage.tsx`, `AutopoiesisGraphPanel.tsx` など）の `animate-ping` / `animate-pulse` を静的かつ上品なグラデーション/ソフトシャドウに変更。
